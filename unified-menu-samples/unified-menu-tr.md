# Birleşik Menü Yapısı (Türkçe)

Bu belge, Krita, Photopea ve Photoshop standartlarına göre sentezlenmiş bir menü yapısını içerir.

| Menü | Öğe | Açıklama | İpucu (Tooltip) | Kısayol |
| :--- | :--- | :--- | :--- | :--- |
| **Dosya** | Yeni | Yeni belge oluşturur. | Boş bir belge veya şablondan yeni proje başlatır. | `Ctrl+N` |
| | Aç... | Mevcut dosyayı açar. | Bilgisayardan veya buluttan dosya seçip açar. | `Ctrl+O` |
| | Son Kullanılanlar | Son çalışılan dosyalar. | Yakın zamanda üzerinde çalışılan dosyaları listeler. | - |
| | Kapat | Aktif belgeyi kapatır. | Mevcut sekmeyi veya belgeyi kapatır. | `Ctrl+W` |
| | Tümünü Kapat | Tüm belgeleri kapatır. | Çalışma alanındaki tüm açık belgeleri kapatır. | `Ctrl+Alt+W` |
| | Kaydet | Aktif belgeyi kaydeder. | Mevcut dosyadaki değişiklikleri kaydeder. | `Ctrl+S` |
| | Farklı Kaydet... | Yeni adla/formatla kaydeder. | Belgenin bir kopyasını farklı isim veya türde kaydeder. | `Ctrl+Shift+S` |
| | İçe Aktar... | Varlık ekler. | Resim, fırça veya palet gibi dosyaları projeye dahil eder. | - |
| | Dışa Aktar... | Yaygın formatlara aktarır. | Projeyi PNG, JPG veya web formatlarında kaydeder. | - |
| | Yazdır... | Belgeyi yazdırır. | Belgeyi yazıcıya gönderir. | `Ctrl+P` |
| | Çıkış | Uygulamayı kapatır. | Programdan tamamen çıkar. | `Ctrl+Q` |
| **Düzen** | Geri Al | Son işlemi geri alır. | En son yapılan değişikliği iptal eder. | `Ctrl+Z` |
| | Yinele | Geri alınan işlemi uygular. | Az önce geri alınan işlemi tekrar uygular. | `Ctrl+Shift+Z` / `Ctrl+Y` |
| | Kes | Seçimi panoya keser. | Seçili alanı kaldırır ve panoya kopyalar. | `Ctrl+X` |
| | Kopyala | Seçimi panoya kopyalar. | Seçili alanı panoya kopyalar. | `Ctrl+C` |
| | Yapıştır | Pano içeriğini yapıştırır. | Pano içeriğini mevcut katmana veya yeni katmana yerleştirir. | `Ctrl+V` |
| | Özel Yapıştır | Özelleştirilmiş yapıştırma. | Seçimin içine, aynı koordinata veya yeni katman olarak yapıştırır. | `Ctrl+Shift+V` |
| | Doldur... | Renk veya desenle doldurur. | Mevcut seçimi veya katmanı düz renk/desenle doldurur. | `Shift+F5` |
| | Kontur... | Seçim etrafına çizgi çeker. | Seçimin kenarlarına bir çerçeve (ana hat) ekler. | - |
| | Serbest Dönüştürme | Seçimi/Katmanı dönüştürür. | Seçimi ölçeklendirir, döndürür veya deforme eder. | `Ctrl+T` |
| | Tercihler | Uygulama ayarları. | Çalışma alanı, performans ve arayüz ayarlarını yapılandırır. | `Ctrl+K` |
| **Görüntü** | Mod | Renk modunu değiştirir. | RGB, CMYK veya Gri Tonlama arasında geçiş yapar. | - |
| | Ayarlamalar | Renk ve ışık ayarları. | Seviyeler, Eğriler, Pozlama ve Doygunluk ayarlarına erişir. | - |
| | Otomatik Ton | Otomatik ton ayarı. | Siyah ve beyaz noktaları otomatik olarak ayarlar. | `Ctrl+Shift+L` |
| | Otomatik Kontrast | Otomatik kontrast ayarı. | Genel kontrastı otomatik olarak dengeler. | `Ctrl+Alt+Shift+L` |
| | Görüntü Boyutu... | Piksel boyutunu değiştirir. | Resmin boyutlarını ve çözünürlüğünü ayarlar. | `Ctrl+Alt+I` |
| | Tuval Boyutu... | Çalışma alanını değiştirir. | Resmi boyutlandırmadan çalışma alanını genişletir veya daraltır. | `Ctrl+Alt+C` |
| | Görüntü Döndürme | Tuvali döndürür. | Tüm resmi 90, 180 derece döndürür veya ters çevirir. | - |
| | Kırp | Resmi kırpar. | Resmin kenarlarını seçime göre temizler. | - |
| **Katman** | Yeni Katman | Boş katman oluşturur. | Projeye yeni bir boş katman ekler. | `Ctrl+Shift+N` |
| | Yeni Grup | Katman klasörü oluşturur. | Katmanları düzenlemek için bir grup/klasör oluşturur. | `Ctrl+G` |
| | Katmanı Çoğalt | Aktif katmanı kopyalar. | Seçili katmanın birebir kopyasını oluşturur. | `Ctrl+J` |
| | Katmanı Sil | Aktif katmanı siler. | Seçili katmanı kalıcı olarak kaldırır. | - |
| | Katman Stili | Dekoratif efektler ekler. | Gölge, Eğim, Kontur veya Işıma gibi efektler ekler. | - |
| | Akıllı Nesne | Yıkıcı olmayan nesne. | Katmanı, kalitesini korumak için akıllı nesneye dönüştürür. | - |
| | Yerleştir | Sıralamayı değiştirir. | Katmanları öne, arkaya, en öne veya en arkaya taşır. | - |
| | Hizala/Dağıt | Katmanları hizalar. | Katman kenarlarını hizalar veya eşit aralıklarla dağıtır. | - |
| | Katmanları Birleştir | Seçili katmanları birleştirir. | Seçili katmanları tek bir katmanda toplar. | `Ctrl+E` |
| | Görüntüyü Düzleştir | Tüm katmanları birleştirir. | Tüm görünür katmanları tek bir arka plan katmanına indirger. | - |
| **Seç** | Tümünü Seç | Tüm tuvali seçer. | Belgenin tamamını kapsayan bir seçim oluşturur. | `Ctrl+A` |
| | Seçimi Kaldır | Seçimi temizler. | Mevcut seçim çerçevesini kaldırır. | `Ctrl+D` |
| | Yeniden Seç | Son seçimi geri getirir. | En son yapılan seçimi tekrar aktif eder. | `Ctrl+Shift+D` |
| | Seçimin Tersini Seç | Tersini seçer. | Mevcut seçimin dışında kalan her şeyi seçer. | `Ctrl+Shift+I` |
| | Tüm Katmanları Seç | Tüm katmanları seçer. | Katman panelindeki her şeyi seçili hale getirir. | `Ctrl+Alt+A` |
| | Renk Aralığı... | Renge göre seçer. | Belirli renk tonlarına göre seçim oluşturur. | - |
| | Seç ve Maskele... | Kenarları iyileştirir. | Kenarları yumuşatmak ve iyileştirmek için gelişmiş arayüzü açar. | `Ctrl+Alt+R` |
| | Değiştir | Seçim sınırlarını değiştirir. | Seçimi genişletir, daraltır, yumuşatır veya kenarlarını inceltir. | - |
| **Filtre** | Son Filtre | Son filtreyi tekrarla. | Kullanılan son filtreyi aynı ayarlarla tekrar uygular. | `Ctrl+Alt+F` |
| | Filtre Galerisi | Yaratıcı filtreler. | Sanatsal ve yapısal filtreleri önizler ve uygular. | - |
| | Neural Filters | AI destekli filtreler. | Cilt pürüzsüzleştirme gibi karmaşık işlemler için yapay zeka kullanır. | - |
| | Sıvılaştır... | Deformasyon araçları. | Piksel itme, çekme ve bükme gibi şekillendirme araçlarını açar. | `Ctrl+Shift+X` |
| | Bulanıklaştır | Görüntüyü yumuşatır. | Gauss, Hareket veya Radyal bulanıklık uygular. | - |
| | Deforme Et | Geometrik bozukluklar. | Dalga, Dalgalanma veya Küreselleştirme efektleri ekler. | - |
| | Parazit | Kumlanma kontrolleri. | Görüntüye kumlanma ekler veya mevcut olanı azaltır. | - |
| | Keskinleştir | Kenarları netleştirir. | Kenar kontrastını artırarak ayrıntıları belirginleştirir. | - |
| | Stilize Et | Sanatsal efektler. | Kabartma, Rüzgar veya Kenar Işıması gibi efektler uygular. | - |
| **Görünüm** | Yakınlaştır | Büyütme oranını artırır. | Belgeye yaklaşır. | `Ctrl++` |
| | Uzaklaştır | Büyütme oranını azaltır. | Belgeden uzaklaşır. | `Ctrl+-` |
| | Ekrana Sığdır | Ekranı kaplar. | Belgenin ekrana tam sığacak şekilde görüntülenmesini sağlar. | `Ctrl+0` |
| | Gerçek Pikseller | %100 görünüm. | Belgeyi orijinal boyutunda görüntüler. | `Ctrl+1` |
| | Ekstralar | Yardımcıları aç/kapat. | Kılavuz, ızgara ve seçim sınırlarını topluca gösterir/gizler. | `Ctrl+H` |
| | Göster | Yardımcı öğeler. | Izgara, katman sınırları veya kılavuzlar gibi öğeleri seçerek gösterir. | - |
| | Cetveller | Cetvelleri aç/kapat. | Ölçüm cetvellerini gösterir veya gizler. | `Ctrl+R` |
| | Yapıştır | Mıknatıs özelliği. | Kılavuzlara, ızgaraya ve katmanlara yapışmayı sağlar. | `Ctrl+Shift+;` |
| **Pencere** | Çalışma Alanı | Arayüz düzenini değiştirir. | Boyama, Tasarım gibi hazır arayüz düzenlerinden birini seçer. | - |
| | Eylemler | Otomatik işlemler. | Sık yapılan işlemlerin kaydedilmesini ve çalıştırılmasını sağlar. | - |
| | Geçmiş | İşlem geçmişi. | Yapılan tüm adımları listeler ve önceki hallerine dönmeyi sağlar. | - |
| | Katmanlar | Katmanlar paneli. | Katman yönetim panelini açar veya kapatır. | `F7` |
| | Kanallar | Renk kanalları. | Renk bileşenleri (Kırmızı, Yeşil, Mavi) panelini açar. | - |
| | Yollar | Yol paneli. | Vektör yollarının yönetim panelini açar. | - |
| | Renk | Renk seçici. | Renk seçme ve palet panelini açar. | - |
| | Fırçalar | Fırça ayarları. | Fırça motoru ve ön ayarları panelini açar. | - |
| | Özellikler | Öğe özellikleri. | Seçili katman veya aracın detaylarını görüntüler. | - |
| **Yardım** | Hakkında | Program bilgisi. | Sürüm, lisans ve yapımcı bilgilerini görüntüler. | - |
| | Eğitimler | Öğrenme merkezi. | Resmi eğitim rehberlerine ve videolarına erişir. | - |
| | Kısayollar | Klavye haritası. | Klavye kısayollarını görüntüler ve özelleştirir. | - |
