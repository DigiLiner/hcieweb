# Unified Menu Structure (English)

This document contains a synthesized menu structure based on Krita, Photopea, and Photoshop standards.

| Menu | Item | Description | Tooltip | Shortcut |
| :--- | :--- | :--- | :--- | :--- |
| **File** | New | Create a new document. | Create a new blank document or from template. | `Ctrl+N` |
| | Open... | Open an existing document. | Open a document from local disk or cloud. | `Ctrl+O` |
| | Open Recent | Recently used documents. | List and open files you worked on recently. | - |
| | Close | Close the active document. | Close the current tab or document. | `Ctrl+W` |
| | Close All | Close all open documents. | Close all active documents in the workspace. | `Ctrl+Alt+W` |
| | Save | Save the active document. | Save changes to the current file. | `Ctrl+S` |
| | Save As... | Save with a new name/format. | Save a copy of the document with a different name or type. | `Ctrl+Shift+S` |
| | Import... | Import external assets. | Bring images, brushes, or palettes into the project. | - |
| | Export... | Export to common formats. | Save the project as PNG, JPG, or other web formats. | - |
| | Print... | Print the document. | Send the document to a printer. | `Ctrl+P` |
| | Exit | Close application. | Quit the program. | `Ctrl+Q` |
| **Edit** | Undo | Revert the last action. | Undo the most recent change. | `Ctrl+Z` |
| | Redo | Reapply last reverted action. | Redo the move that was just undone. | `Ctrl+Shift+Z` / `Ctrl+Y` |
| | Cut | Remove selection to clipboard. | Cut the selected area and copy it. | `Ctrl+X` |
| | Copy | Copy selection to clipboard. | Copy the selected area. | `Ctrl+C` |
| | Paste | Paste clipboard content. | Place content from clipboard into the layer. | `Ctrl+V` |
| | Paste Special | Specialized paste options. | Paste into selection, as a new layer, or in place. | `Ctrl+Shift+V` |
| | Fill... | Fill with color or pattern. | Fill the current selection or layer with solid color or pattern. | `Shift+F5` |
| | Stroke... | Draw border around selection. | Apply a border (outline) to the selection. | - |
| | Free Transform | Transform selection/layer. | Scale, rotate, skew, or distort the selection. | `Ctrl+T` |
| | Preferences | Application settings. | Configure workspace, performance, and UI settings. | `Ctrl+K` |
| **Image** | Mode | Change color mode. | Switch between RGB, CMYK, or Grayscale. | - |
| | Adjustments | Color & light adjustments. | Access Levels, Curves, Exposure, and Saturation settings. | - |
| | Auto Tone | Automated tonal adjustment. | Automatically adjusts the black and white points. | `Ctrl+Shift+L` |
| | Auto Contrast | Automated contrast adjustment. | Automatically adjusts the overall contrast. | `Ctrl+Alt+Shift+L` |
| | Image Size... | Change pixel dimensions. | Resize the image and adjust resolution. | `Ctrl+Alt+I` |
| | Canvas Size... | Change workspace size. | Expand or shrink the working area without resizing image contents. | `Ctrl+Alt+C` |
| | Image Rotation | Rotate or flip canvas. | Rotate the entire image 90, 180 degrees or flip it. | - |
| | Crop | Trim the image. | Trim the edges of the image to the selection. | - |
| **Layer** | New Layer | Create blank layer. | Add a new empty layer to the project. | `Ctrl+Shift+N` |
| | New Group | Create layer folder. | Create a group to organize multiple layers. | `Ctrl+G` |
| | Duplicate Layer | Copy active layer. | Create an exact copy of the selected layer. | `Ctrl+J` |
| | Delete Layer | Remove active layer. | Delete the selected layer permanently. | - |
| | Layer Style | Decorative effects. | Add Drop Shadow, Bevel, Stroke, or Glow effects. | - |
| | Smart Object | Non-destructive object. | Convert layer into a smart object to preserve quality. | - |
| | Arrange | Change stack order. | Move layers forward, backward, to front, or to back. | - |
| | Align/Distribute | Align layers. | Align layer edges or distribute them evenly. | - |
| | Merge Layers | Combine selected layers. | Merge the selected layers into one. | `Ctrl+E` |
| | Flatten Image | Merge all layers. | Merge all visible layers into a single background layer. | - |
| **Select** | All | Select entire canvas. | Create a selection that covers the whole document. | `Ctrl+A` |
| | Deselect | Clear current selection. | Remove the current selection marquee. | `Ctrl+D` |
| | Reselect | Restore last selection. | Bring back the most recent selection. | `Ctrl+Shift+D` |
| | Inverse | Invert selection. | Select everything that is currently unselected. | `Ctrl+Shift+I` |
| | All Layers | Select all layers. | Select every layer in the layer panel. | `Ctrl+Alt+A` |
| | Color Range... | Select by color. | Create a selection based on specific color samples. | - |
| | Select and Mask... | Refine edges. | Open advanced interface for edge refining. | `Ctrl+Alt+R` |
| | Modify | Change selection bounds. | Expand, Contract, Smooth, or Feather the selection. | - |
| **Filter** | Last Filter | Repeat last filter. | Re-apply the last used filter with same settings. | `Ctrl+Alt+F` |
| | Filter Gallery | Browse creative filters. | Preview and apply various artistic and structural filters. | - |
| | Neural Filters | AI powered edits. | Use artificial intelligence for complex retouching (e.g., skin smoothing). | - |
| | Liquify... | Warping tools. | Push, pull, and warp pixels for advanced reshaping. | `Ctrl+Shift+X` |
| | Blur | Soften image. | Apply Gaussian, Motion, or Radial blur. | - |
| | Distort | Geometric distortion. | Apply Wave, Ripple, or Spherize effects. | - |
| | Noise | Grain controls. | Add or remove grain and artifacts from the image. | - |
| | Sharpen | Enhance edges. | Increase the clarity and detail of image edges. | - |
| | Stylize | Artistic effects. | Apply Emboss, Extrude, or Wind effects. | - |
| **View** | Zoom In | Increase magnification. | Zoom into the document. | `Ctrl++` |
| | Zoom Out | Decrease magnification. | Zoom out from the document. | `Ctrl+-` |
| | Fit on Screen | Fit workspace. | Zoom so the document fills the screen. | `Ctrl+0` |
| | 100% | Actual pixels. | View the document at its actual size. | `Ctrl+1` |
| | Extras | Toggle all aids. | Show or hide all guides, grids, and boundaries. | `Ctrl+H` |
| | Show | Toggle aids. | Toggle specific items like Grid, Layer Bounds, or Guides. | - |
| | Rulers | Toggle rulers. | Show or hide the measurement rulers. | `Ctrl+R` |
| | Snap | Snap to aids. | Toggle snapping to guides, grid, and layers. | `Ctrl+Shift+;` |
| **Window** | Workspace | Change UI layout. | Select from predefined layouts like Painting or Design. | - |
| | Actions | Automated tasks. | Record and run sequences of common tasks. | - |
| | History | Undo/Redo browser. | Browse through previous states and jump back. | - |
| | Layers | Layers panel. | Open or close the Layers management panel. | `F7` |
| | Channels | Color channels. | Open or close the Color Channel panel. | - |
| | Paths | Path panel. | Open or close the Vector Path panel. | - |
| | Color | Color picker. | Open or close the Color selection panel. | - |
| | Brushes | Brush settings. | Open or close the Brush engine and presets. | - |
| | Properties | Item properties. | View details about the selected layer or tool. | - |
| **Help** | About | Program info. | View version, license, and credit information. | - |
| | Tutorials | Learning center. | Access official learning guides and videos. | - |
| | Shortcuts | Keyboard map. | View and customize keyboard shortcuts. | - |
